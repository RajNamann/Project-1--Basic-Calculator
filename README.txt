# Advanced Scientific Calculator  
Created by **Naman R Srivastava**

This is a feature-rich scientific calculator built using **HTML**, **CSS**, and **JavaScript**.  
It supports both basic arithmetic and advanced scientific operations with live expression evaluation, history tracking, memory registers, and angle mode switching between **Degrees (DEG)** and **Radians (RAD)**.

---
link-https://rajnamann.github.io/Basic-calculator/
## Features

### Basic Operations
- Addition, subtraction, multiplication, division
- Decimal input and multi-digit handling
- Backspace and clear controls (`CE`, `AC`)

### Scientific Functions
- `sin`, `cos`, `tan` (supports **DEG/RAD** mode)
- `log` (log base 10)
- `ln` (natural log)
- `√` (square root), cube root, powers: `x²`, `x³`, `xʸ`
- `1/x` (reciprocal)
- `n!` (factorial)
- Constants: **π**, **e**

### Additional Capabilities
- `DEG / RAD` toggle
- Smart expression preview while typing
- Interactive history (click result to reuse)
- Memory functions: `MC`, `MR`, `M+`, `M-`
- Keyboard input support

| Operation     | Input    | Result |
| ------------- | -------- | ------ |
| Addition      | 2+3      | 5      |
| Power         | 2**3     | 8      |
| Square Root   | √9       | 3      |
| Trigonometric | sin(30)  | 0.5    |
| Logarithm     | log(100) | 2      |
